# GT-MARKDAWIN v2.0 🚀
**الإطلاق الرسمي: "مارك دَوِّنْ" - محرر ماركداون عصري بدعم متكامل للعربية**

<img src="gt-markdawin-icon.png" width="256" height="256" alt="أيقونة التطبيق">

بكل فخر وسرور، نعلن عن الإصدار الثاني من **GT-MARKDAWIN** - محرر ماركداون متكامل صُمم خصيصًا لتلبية احتياجات الكتّاب والمطورين الناطقين بالعربية، بميزات متقدمة ودعم كامل للنصوص من اليمين لليسار (RTL).

## ✨ الميزات الرئيسية

### 🖋️ تحرير متقدم بدعم عربي
- **دعم RTL/LTR متكامل**: واجهة مصممة خصيصًا للغتين العربية والإنجليزية مع تبديل سلس
- **مزامنة ذكية**: تمرير متزامن ودقيق بين نافذة التحرير ونافذة المعاينة
- **إحصائيات حية**: عدّاد كلمات وحروف وأسطر فوري مع دعم خاص للغة العربية
- **لوحة إيموجي شاملة**: مجموعة منظمة من الرموز التعبيرية بما فيها أعلام الدول العربية والإسلامية
- **أدوات تحرير كاملة**: تراجع/إعادة، إدراج فيديو، صور متحركة، وجميع عناصر ماركداون

### 🎨 تخصيص شامل وإدارة أصول
- **نظام الخطوط المدمج**: دعم كامل للخطوط العربية المخصصة (خطوط قرآنية، خطوط عربية حديثة)
- **مكتبة إيموجي شاملة**: دعم للصور بدقة 72x72 وSVG
- **تصدير مرن**: حفظ كمستند ماركداون خام أو كملف HTML كامل ومستقل أو PDF
- **تحكم كامل في التخطيط**: تكبير المحرر أو المعاينة أو العرض المتساوي

### 🌐 منصات متعددة وتشغيل مرن
- **حزمة AppImage لأنظمة Linux**: تطبيق موحد يعمل على معظم التوزيعات
- **بيئة Python مدمجة**: خادم ويب محلي مضمّن للتشغيل الفوري
- **تطبيق ويب تقدمي (PWA)**: يعمل دون اتصال بالإنترنت مع تجربة مستخدم محسنة

## 🚀 التشغيل السريع

### 📥 تحميل وتشغيل AppImage (مستحسن)

```bash
# تحميل التطبيق
wget https://github.com/SalehGNUTUX/GT-MARKDAWIN/releases/download/v2.0/GT-MARKDAWIN-v2.0-x86_64.AppImage
```
# جعله قابلاً للتنفيذ
```
chmod +x GT-MARKDAWIN-v2.0-x86_64.AppImage
```
# التشغيل
```
./GT-MARKDAWIN-v2.0-x86_64.AppImage
```
🛠️ البناء من المصدر
bash
# استنساخ المستودع
```
git clone https://github.com/SalehGNUTUX/GT-MARKDAWIN.git
```
```
cd GT-MARKDAWIN
```
# تشغيل التطبيق مباشرة
```
./run_app.sh
```
# أو بناء حزمة AppImage
```
./build\ appimage\ package.sh
```
📦 بناء حزمة AppImage
المتطلبات المسبقة
bash
# تأكد من تثبيت Python3
```
python3 --version
```
# تحميل أدوات البناء (إذا لم تكن موجودة)
```
wget https://github.com/linuxdeploy/linuxdeploy/releases/download/continuous/linuxdeploy-x86_64.AppImage
```
```
wget https://github.com/AppImage/AppImageKit/releases/download/continuous/appimagetool-x86_64.AppImage
```
```
chmod +x linuxdeploy-x86_64.AppImage appimagetool-x86_64.AppImage
```
خطوات البناء
bash
# 1. جعل سكريبت البناء قابلاً للتنفيذ
```
chmod +x "build appimage package.sh"
```
# 2. تشغيل سكريبت البناء
```
./build\ appimage\ package.sh
```
# 3. الملف الناتج: GT-MARKDAWIN-v2.0-x86_64.AppImage

🔧 التخصيص
إضافة خطوط مخصصة
ضع ملفات الخطوط في مجلد emojis/fonts/

```
json
{
  "name": "اسم الخط",
  "path": "emojis/fonts/خطك.ttf"
}
```

أضف الصور في emojis/72x72/ أو emojis/svg/

شغّل ./emojiscript.sh لتحديث emojis.json
